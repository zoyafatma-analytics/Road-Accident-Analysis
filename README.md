# 🚦 Road Accident Analysis Dashboard

## 📝 Project Overview
The **Road Accident Analysis Dashboard** is an interactive Power BI project developed to analyze road accident data and uncover insights related to casualties, accident severity, vehicle involvement, road conditions, and geographical distribution.

This dashboard helps in understanding accident patterns, identifying high-risk factors, and supporting data-driven road safety analysis through dynamic visualizations and KPIs.

---

## 🎯 Objective
The objective of this project is to:
- Analyze accident and casualty trends
- Identify major causes and conditions contributing to accidents
- Understand casualty distribution by vehicle type and road type
- Compare urban and rural accident patterns
- Support road safety awareness and decision-making

---

## 📌 Key Features
- 🚗 **Vehicle-wise Casualty Analysis**
- 📈 **Monthly Casualty Trend Comparison**
- 🛣️ **Road Type-Based Casualty Distribution**
- 🌗 **Light Condition Analysis (Day vs Night)**
- 🌍 **Urban vs Rural Accident Analysis**
- 🗺️ **Geographical Accident Mapping**
- 🔎 Interactive filters for:
  - Road Surface
  - Weather Conditions

---

## 📂 Dataset Information
The dashboard uses an Excel dataset containing:
- Accident Date
- Vehicle Type
- Casualty Count
- Accident Severity
- Road Type
- Weather Conditions
- Light Conditions
- Urban/Rural Area
- Geographical Location Data

---

## 🛠️ Tools & Technologies Used
- **Power BI** – Dashboard Development & Data Visualization
- **Microsoft Excel** – Data Source
- **Power Query** – Data Cleaning & Transformation
- **DAX** – KPI and Measure Calculations

## 📈 Insights Generated
- Majority of casualties were associated with cars
- Urban areas recorded higher accident rates compared to rural areas
- Daytime accidents were significantly higher than nighttime accidents
- Single carriageway roads showed the highest number of casualties
- Weather and road surface conditions influenced accident frequency
